无法回答“有哪些 API 正在对外提供服务”，就很难讨论它们是否安全。

## 汇总多种来源

网关配置、代码路由、流量日志和前端资源看到的是不同视角，应进行交叉验证。

## 标记生命周期

为接口记录负责人、数据等级、认证方式、版本和下线日期。无人负责的接口应被视为风险，而不是普通缺项。

## 持续发现变化

把资产发现接入发布流程，对新增外部入口和认证方式变化生成可审计事件。
